# Kuratierte BGH-Leitsätze

Public-Domain-Auszüge aus rechtsprechung-im-internet.de (BMJ),
manuell verifiziert von Sonnet-Recherche-Agents mit WebFetch.

Format: `{lawId}_{section}.json` — wird vom CitationDrawer
automatisch geladen, wenn ein:e Anwält:in auf einen verifizierten
§-Badge klickt.

Stand: April 2026. Wöchentliche Aktualisierung geplant.
